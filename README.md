# EastBridge Technologies Website v11.1

Production-oriented static multilingual website for:

- `eastbridge.vn` — Vietnamese default at `/`; English under `/en/`
- `eastbridge.my` — Malay under `/ms/`
- `eastbridge.sg` — Simplified Chinese under `/zh/`

## Main features

- 44 fully rendered language/page combinations
- Corresponding-page language switching
- Cross-domain `hreflang`, canonical links and `x-default`
- Domain-specific sitemaps
- GitHub Pages workflow and `CNAME`
- Cloudflare Pages headers, redirects and optional domain-routing Worker
- Vietnamese-local CSR visual treatment
- Responsive navigation and corrected mobile hero buttons
- Subtle animation, light sweep, text gradient, reveal motion and reduced-motion support
- No framework or build step required

Read `DEPLOYMENT_GUIDE.md` before going live.
