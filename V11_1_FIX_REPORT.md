# EastBridge Technologies v11.1 Fix Report

## Completed fixes

1. Hero bridge visual now uses a PNG primary asset on all four homepages, with automatic WebP-to-PNG fallback for static hosting.
2. Hero title, eyebrow, copy and call-to-action labels now change with each slide. The previous selector was matching slide data containers instead of the visible content elements.
3. Supporting images have controlled maximum widths and heights on desktop, tablet and mobile while preserving adequate visual impact.
4. CSR uses a dedicated Vietnam-inspired red/gold digital visual, enhanced border lighting, animated gold highlights and restrained national styling.
5. Multilingual blog pages added in Vietnamese, English, Malay and Simplified Chinese, together with editable no-index post templates.
6. Blog links added to navigation and footer areas, and sitemap entries added for each regional language URL.
7. Existing v10 pages and content headings were reviewed against v11. The current release retains the broader service, solution, training, advisory, industry and CSR coverage while using the refined multilingual information architecture.
8. JavaScript syntax, XML sitemap parsing and local internal asset/link resolution were validated.
